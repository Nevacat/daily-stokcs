# 테스트 규칙 (테스트 파일을 다룰 때만 적용)

- 새 기능에는 반드시 단위 테스트를 함께 작성한다.
- 뉴스 수집/분석 로직은 외부 API를 목(mock) 처리하고 테스트한다 — 실제 API 호출 금지.
- 추천 점수 계산 로직은 경계값 테스트 필수 (뉴스 0건, 전부 부정, 동점 등).
- 테스트 파일 위치: 소스 파일과 같은 폴더에 `*.test.ts` (colocation).
- 커밋 전 전체 테스트가 통과해야 한다.
