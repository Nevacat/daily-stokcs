# 보안 점검 체크리스트

## 시크릿
- [ ] API 키, 토큰, 비밀번호가 코드/설정에 하드코딩되어 있지 않은가
- [ ] .env 파일이 .gitignore에 포함되어 있는가
- [ ] 로그에 시크릿/개인정보가 출력되지 않는가

## 입력 검증
- [ ] 외부 입력(쿼리 파라미터, 바디)에 검증 스키마가 있는가
- [ ] SQL/NoSQL 인젝션 가능성 (raw query 사용처 확인)
- [ ] 외부 뉴스 데이터(제목/본문)를 렌더링할 때 XSS 방어가 되는가

## 인증/인가
- [ ] 보호가 필요한 엔드포인트에 인증 미들웨어가 적용되어 있는가
- [ ] 다른 사용자의 리소스(관심 종목 등)에 접근 가능한 IDOR가 없는가

## 외부 통신
- [ ] 외부 API 호출에 타임아웃/재시도 제한이 있는가
- [ ] 외부 URL로 리다이렉트 시 오픈 리다이렉트 가능성이 없는가

## 의존성
- [ ] 새로 추가된 패키지에 알려진 취약점이 없는가 (`npm audit`)
